# Week 2: Theme Toggle Application

A dynamic web page that allows users to seamlessly switch between light and dark themes using vanilla JavaScript.

## 🚀 Live Demo
*   [Click here to view my live Theme Toggle website](https://[YOUR-ORGANIZATION-NAME].github.io/week2_theme_toggle/)

## 🛠️ Features
*   **Smooth Theme Switching:** Toggles background and text colors using CSS variables.
*   **Clean Architecture:** Separated HTML structure, CSS layouts, and JavaScript logic.
*   **Responsive Design:** Fully responsive and optimized for mobile or desktop viewports.

## 📁 Files Included
*   `index.html` - The webpage structure and layout.
*   `style.css` - Custom styling utilizing CSS custom properties for color tokens.
*   `script.js` - JavaScript event listener handling user click events.
